About Pattern Sense

Pattern Sense is a deep learning-powered solution that automatically classifies fabric patterns such as Animal, Floral, Geometric, and more. This project helps designers, quality controllers, and decorators instantly recognize pattern types using image-based AI detection.

Built using TensorFlow, Keras, Flask, and Tailwind CSS, our goal is to improve efficiency across fashion, textile, and design industries. Our model uses MobileNetV2 with transfer learning, achieving real-time inference and high accuracy from user-uploaded images.
